### all()
```
itarbale = [1, 2, 3, 4]
if all(iterable):
    # all element in the iterable should be true in order
    # to get inside the if condition, if even one element
    # is false it will not get in here
```
### any
```
iterable = [1, 2, 3, 4, []]
if any(iterabel):
    # if any of the iterable element is true it will get in here
```

### id()

