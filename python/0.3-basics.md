## false producing expression:
**any empty dataType, 0 or None compiled to false:**
```
string = ""
bool(string) # false
zero = 0
var = None
listy = []
bool(listy) # flase
dict = {}
tup = ()
```
**all of these above if put in an if condition**
**will be interpreted to false**
