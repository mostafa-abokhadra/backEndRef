### most common used Math methods
- round()
- ceil()
- floor() 
- min()
- max()
- pow()
- random()
- trunc()

```js
Math.round(99.2) // 99
Math.round(99.5) // 100
Math.ceil(99.2)  // 100
Math.floor(99.2) // 99
Math.min(1, 2, 3, 4, 5) // 1
Math.max(1, 2, 3, 4, 5) //  5
Math.pow((2, 4)) // 16
Math.random() // random number
Math.trunc(99.5) // return the integer part -> 99
```
